/

Bartender is an app that will mix a drink by turning on a set of hardware pumps

pump[…] - direct access to hardware pumps liquids string[] - all the unique liquids found across all recipes current{…} - current drink recipe in progress recipe[…] - available list of drinks known to this bartender

/pump

direct access to hardware pumps

id int32 - ids from 0-8 for the installed pumps gpioPin int32 - Raspberry PI GPIO pin number. Do not change unless you're rewired your Pi. timeToVolumeRatioMs int32 - number of millisecs to turn on pump to pour one milliliter liquid string - name of the liquid. If name does not match (case sensitive) recipe ingredient exactly, it will not recognized as same liquid.

/pump={id}/on

turn on the pump

/pump={id}/off

turn off the pump

/current

current drink recipe in progress

percentComplete int32 - 0-100 percent complete with drink complete boolean - true if drink has finished being made auto[…] - ingredients from recipe that are available in the pumps and would be poured automatically manual[…] - ingredients from recipe that are not available in the pumps and would need to be poured manually by user

/current/stop

Stop making drink where ever in the process it is

/current/update

percentComplete int32 - 0-100 percent of how far along is this drink done making complete boolean - is this drink ready for drinking
manual[…] - ingredients that should be poured manually
id int32 - unique id for this ingredient
complete boolean - is this ingredient done pouring
auto[…] - ingredients that should be poured automatically
id int32 - unique id for this ingredient
complete boolean - is this ingredient done pouring
percentComplete int32 - 0-100 percent of how far along is this ingredient is done pouring

/current/auto

ingredients from recipe that are available in the pumps and would be poured automatically

id int32 - unique id for this ingredient ingredient{…} - liquid ingredient complete boolean - is this liquid done pouring percentComplete int32 - what percentage is this liquid complete pouring

/current/auto={id}/ingredient

liquid ingredient

liquid string - liquid name. If name does not match (case sensitive) pump liquid exactly, it will not recognized as same liquid. amount decimal64 - in ounces of liquid weight int32 - in grams of liquid determined by amount and estimate of grams per ounce

/current/manual

ingredients from recipe that are not available in the pumps and would need to be poured manually by user

id int32 - unique id for this ingredient ingredient{…} - liquid ingredient complete boolean - is this liquid done pouring

/current/manual={id}/done

mark step done

/current/manual={id}/ingredient

liquid ingredient

liquid string - liquid name. If name does not match (case sensitive) pump liquid exactly, it will not recognized as same liquid. amount decimal64 - in ounces of liquid weight int32 - in grams of liquid determined by amount and estimate of grams per ounce

/recipe

available list of drinks known to this bartender

id int32 - unique id for this recipe name string - name of recipe description string - description of recipe ingredient[…] - ingredients for recipe

/recipe={id}/make

make this drink

multiplier decimal64 - Optional, make it a double by passing 2

/recipe={id}/ingredient

ingredients for recipe

liquid string - liquid name. If name does not match (case sensitive) pump liquid exactly, it will not recognized as same liquid. amount decimal64 - in ounces of liquid weight int32 - in grams of liquid determined by amount and estimate of grams per ounce
